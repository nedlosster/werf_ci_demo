# Демо

> **Доставка по контракту: два стека -- один `werf converge`**
> Разработка прямо в dev-кластере, где dev неотличим от prod.
> Дистилят практики Kubernetes 2020-2025.

Раздел содержит материалы для показа репозитория: план доклада (~64 минуты),
слайд-аутлайн и банк вопросов с ответами. Опирается на остальные разделы
документации и служит сценарием для презентации связки kube_ci с продуктами.

Доклад построен как дистилляция опыта разработки приложений под Kubernetes за
2020-2025 годы. Ядро часа -- три темы: развязка контрактом `.helm/def.sh` плюс
werf converge, разработка прямо в кластере с живым циклом работы на app1, операции
доставки на живых командах. Блок «Контекст» держит три слайда: компромиссы с DORA
и версиями, интеграция с внешними CI, хранение секретов.

## Статьи

- [talk-plan-60min.md](talk-plan-60min.md) -- поминутная раскладка доклада
  (~64 минуты): одиннадцать блоков с таймингом, целью, тезисами и ссылками на
  статьи для углубления. Около 56 минут материала и 8 минут вопросов. Включает
  мини-сценарий живой демонстрации цикла работы на фронте app1.
- [slides.md](slides.md) -- аутлайн вспомогательных слайдов: восемнадцать слайдов
  по блокам доклада 1-10, каждый с тремя-пятью телеграфными тезисами и строкой
  «Показать» со ссылкой на диаграмму или профильную статью. Для блока вопросов --
  backup-слайды из банка вопросов.
- [qa-bank.md](qa-bank.md) -- вопросы и ответы, сгруппированные по темам, со
  ссылками на профильные статьи. Отдельная группа -- разработка в кластере
  (dev-in-cluster против docker-compose, отладка k8s-логики, кеши и волумы при
  рестарте, `git pull` по ключу в поде, цена dev-подов). Покрывает острые
  компромиссы схемы: insecure-registry, loose-giterminism, общий кластер dev/prod,
  nip.io, ключ werf, dev-под с ssh-ключом.

## Слайды в HTML

Две HTML-сборки плана и слайдов. Содержание -- производное от slides.md и
talk-plan-60min.md; при правке тезисов синхронизировать оба файла. Диаграммы из
[pics/](../pics/) и ссылки на исходники в GitHub встроены прямо в слайды.

- [talk-deck.html](talk-deck.html) -- полноэкранная презентация под проектор:
  один слайд на экран, навигация вперёд/назад (кнопки, стрелки, Space, F -- во
  весь экран). Титульный слайд с автором, 18 слайдов по блокам, закрывающий слайд
  с контактами. Всего 20 слайдов. В правой части каждого слайда -- встроенная
  скроллируемая страница документации (из [embed/](embed/), генерируется
  `scripts/demo/build-embeds.sh`), внизу -- подсвеченная ссылка на эту страницу в
  GitHub; где уместно, под тезисами слева -- диаграмма из [pics/](../pics/).
  Прямое встраивание github.com невозможно (X-Frame-Options), поэтому встроена
  локально отрендеренная копия дока, а ссылка ведёт на GitHub.
- [talk-flow.html](talk-flow.html) -- вертикально-скроллируемая лента-суфлёр:
  разделы и слайды отчёркнуты, плюс цель блока, развёрнутая заметка спикера и
  пошаговый сценарий живой демонстрации (блок 6). По ней ведётся рассказ.
