# Демо

Раздел содержит материалы для показа репозитория: план доклада на 40 минут,
слайд-аутлайн и банк вопросов с ответами. Опирается на остальные разделы
документации и служит сценарием для презентации связки kube_ci с продуктами.

## Статьи

- [talk-plan-40min.md](talk-plan-40min.md) -- поминутная раскладка доклада на 40
  минут: двенадцать блоков с таймингом, целью, тезисами и ссылками на статьи для
  углубления. Сумма блоков -- 40 минут, последние пять под вопросы.
- [slides.md](slides.md) -- аутлайн вспомогательных слайдов: семнадцать слайдов
  по блокам доклада 1-11, каждый с тремя-пятью телеграфными тезисами и строкой
  «Показать» со ссылкой на диаграмму или профильную статью. Для блока вопросов --
  backup-слайды из банка вопросов.
- [qa-bank.md](qa-bank.md) -- двадцать вопросов и ответов, сгруппированных по
  темам, со ссылками на профильные статьи. Покрывает острые компромиссы схемы:
  insecure-registry, loose-giterminism, общий кластер dev/prod, nip.io, ключ
  werf, dev-под с ssh-ключом.
